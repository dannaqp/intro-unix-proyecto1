Introduction:
This report explores the secure configuration of a Debian 13 environment, prioritizing modern cybersecurity best practices. The documentation covers essential technical implementations, including:

Storage & Privacy: LVM management and LUKS disk encryption.

Credential Security: Advanced password management using KeePass.

Data Integrity: Implementation of cryptographic hashes and secure communication via Proton Mail and GPG.

Professional Development: Integration of GPG keys to sign Git commits, ensuring traceability and verification in coding workflows.

Objectives:
This project aligns with the core competencies of the course. Upon completion, the student will demonstrate practical proficiency in:

System Knowledge: Describing the UNIX/Linux operating system and its key differences from other platforms.

File Systems: Mastery of the UNIX/Linux file structure, attributes, and permission systems.

Cryptography (GPG): Using GNU Privacy Guard to encrypt, decrypt, and sign files and digital communications.

Terminal Proficiency: Executing essential commands via the UNIX/Linux terminal with confidence.

Security Fundamentals: Deep understanding of disk encryption, hashing, and password management.

Version Control: Managing repositories with Git and collaborating on platforms like GitHub and Codeberg.
